## Installation

See [Marmalade: Spreadable Elisp](http://marmalade-repo.org/ "Marmalade: Spreadable Elisp") to add Marmalade to your package archives.
Then execute:

    M-x package-install sudden-death

## Usage
Activate region and then execute:

    M-x sudden-death
